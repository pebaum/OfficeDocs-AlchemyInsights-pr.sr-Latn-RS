# <a name="export-user-account-information"></a>Izvoz informacija o korisničkom nalogu

Izvoz informacija o korisničkom nalogu u na Microsoft 365 admin center, uključujući informacije o licenci, smernice za lozinke, blokirane račune, status sinhronizacije AAD i drugo, idi **Korisnici** > [aktivni korisnici](https://go.microsoft.com/fwlink/p/?linkid=834822) stranice, a zatim izaberite **izvoz Korisnici**.
